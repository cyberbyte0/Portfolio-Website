Security Research

Exploring API security, authentication workflows, and secure design principles.

Approach

I follow a structured methodology to uncover security issues:

  1. Map API endpoints & document requests (Burp Suite)
  2. Validate input handling, authorization, and error responses
  3. Check for OWASP API Top 10 issues (BOLA, excessive data exposure, etc.)
  4. Log findings, classify severity, and report responsibly
